Directory Stack Manager (DSM), Version 1.1

b                - go back to previous directory in history
c <context>      - load a new context from disk
cc               - show context comment
cpx <file>       - copy file into $x
d                - show current keys/dirs
dh               - show this help text
ds               - show index and directory history
f                - go forward to next directory in history
g <key>          - go to directory of key
k                - show key of current directory
k [<directory>]  - show key of specified directory
lc               - list saved contexts
pd <dir> [<key>] - push to directory and set its key
rmk <key>        - remove key/dir pair
sc <context>     - save context to disk
x                - sets $x to current directory (see cpx)
